理解同源策略

发表于 2019-10-18 分类于 http

何为同源

两个页面的协议、域名、端口都相同的情况下，这两个页面就属于同源。

  1. 协议一般是http或者http;

  2. 域名一般表现为www.baidu.com；

  3. 端口在域名后面加上:4000，表示4000端口，如果没有注明端口，则表示默认端口。http默认端口为80，https默认端口为443;
